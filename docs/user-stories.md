# User Stories

### New users
- As a new user, I want to sign in with my university credentials,so that I don’t create yet another password and my student status is verified automatically.

### Regular users
- As a user on the website, I want it to be easy to see where the available drivers are going so that I can pick the best route for me.
- As a user, I want to report someone and block them, so that a bad interaction doesn’t repeat.
- As a user without a car, I want to carpool so that I can get to where I want to go quickly without having to spend an exorbitant amount of money.
- As a user, I want my rating to affect my visibility in matching, so that my good behavior is rewarded. 

### Riders
- As a rider, I want to rate a driver after the trip, so that others can see who is good and who is not so much.
- As a rider, I want the app to handle the transaction so that I can get in and out seamlessly.
- As a rider, I want to see the estimated cost split before I join, so that there are no surprises
- As a rider, I want to see that a driver is a verified student from my, or a local, institution, so that I feel safer getting in the car.

### Drivers
- As a driver, I want to carpool with other people so that I can get gas money and reduce my overall cost to get to my destination. 🤑

### Developers / Administration
- As an admin on the app, I want to be able to review reported users and suspend accounts, so that I can act on safety issues if need be.
